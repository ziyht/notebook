# vue

[官方文档](https://cn.vuejs.org/v2/guide/installation.html)

### 安装

```sh
brew install node

# 安装 淘宝 npm 镜像
# 这样就可以使用 cnpm 命令来安装模块了：cnpm install [name]
npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm install -g vue

cnpm install -g @vue/cli
```



### 创建项目

```sh
vue creat myserver

cd myserver

npm run serve
```
